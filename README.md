## 🚀 Proyectos Destacados

### 🔤 [B2B Document Translator](https://github.com/guille123giles-cloud/b2b-document-translator)
> Plataforma serverless de traducción masiva de documentos (PDF, DOCX, imágenes) con OCR de alta precisión y costo operativo **cero**.

- Stack 100% gratuito construido como alternativa real a Azure Document Translator (servicio de pago)
- Integración con **Google Gemini 2.5 Flash** para OCR contextual y **Google Translate** para traducción
- Reconstrucción inteligente del layout del documento original (estilos, tablas, jerarquía visual)
- Procesamiento en memoria para garantizar privacidad total de los datos

`Python` `Gemini 2.5 Flash` `pdfplumber` `ReportLab` `python-docx` `Streamlit` `Serverless`

---

### 📝 [AI Note Digitizer Pro](https://github.com/guille123giles-cloud/ai-note-digitizer)
> Digitalizador de apuntes manuscritos que va más allá del OCR tradicional: entiende contexto y caligrafía compleja.

- Usa **Google Gemini 1.5 Flash** (visión generativa) para transcribir, estructurar y corregir apuntes
- Mantiene la jerarquía visual del original y produce texto limpio y estructurado
- Interfaz simple construida con Streamlit para uso inmediato sin instalación compleja

`Python` `Gemini 1.5 Flash` `Computer Vision` `Streamlit` `Generative AI`

---

### 🔊 [OCR Async TTS Engine](https://github.com/guille123giles-cloud/ocr-async-tts-engine)
> Motor asíncrono que convierte documentos y fotos de apuntes en audiolibros MP3.

- Arquitectura **asíncrona con Asyncio** para procesamiento paralelo de fragmentos de audio
- Pipeline completo: imagen → texto (Tesseract OCR) → audio (Microsoft Edge TTS)
- Herramienta de accesibilidad real: respuesta inmediata sin esperar a que termine el documento completo

`Python` `Asyncio` `Tesseract OCR` `Edge-TTS` `Multi-threading` `Streamlit`
